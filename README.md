# Flow
Represent user interactions as maps.

## Development
### Get source code
```
git clone https://github.com/codito/flow.git
cd flow
```

### Get bower components
```
npm install -g bower
bower install
```

### Launch the app!
```
mkvirtualenv flow
pip install -r development.txt
python app.py
```

Open [http://localhost:5000]() in a browser.
